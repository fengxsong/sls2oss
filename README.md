# sls2oss

Tool for migrate logs in aliyun sls to oss bucket.

## Pre-requisites

- golang >= 1.15
- make

## Build

```bash
make bin
```

## Run

```bash
./build/_output/bin/sls2oss-linux-amd64 --help
```
